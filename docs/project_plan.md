# Project Plan: Robust Detection of AI-Generated Text Under Paraphrasing

- **Week 1-2 (Completed):** 
    - Brainstorm ideas, define research question.
    - Understand relevance, read papers.
    - Identify gaps, finalize domain, baselines, and metrics.
    - Design paraphrasing methodology.

- **Week 3 (Next):** 
    - Initialize GitHub repository (Current).
    - Create project structure (Current).
    - Collect human-written dataset.
    - Generate AI-written dataset.

- **Week 4:** 
    - Implement TF-IDF + Logistic Regression.
    - Build evaluation pipeline.

- **Week 5:** 
    - Fine-tune an NLP model (e.g., BERT/RoBERTa).
    - Compare performance with baseline.

- **Week 6:** 
    - Implement paraphrasing method.
    - Generate paraphrased dataset split.

- **Week 7:** 
    - Measure performance degradation under paraphrasing.
    - Generate plots and confusion matrices.

- **Week 8:** 
    - Compare models under different levels of paraphrasing.
    - Perform error analysis.

- **Week 9:** 
    - Analyze feature contributions.
    - Potential model optimizations.

- **Week 10:** 
    - Final write-up and documentation.
